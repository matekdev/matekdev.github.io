JavaScript Chip8 emulator

Chip8 is an interpreted programming language. As a part of CMPT 276, we were tasked with designing a Chip8 interpreter that runs in JavaScript. The tasks I completed during the duration of this project were...

- Fully created the entire HTML page for both the emulator and tool

- Created the base structure for the emulator itself (the classes, how the emulator cycled, emulator variables, etc)

- Implemented various opcodes

- Created test suites using Jest for most opcodes

- Entirely created the tool

- Played the role of GitHub master for the duration of the project
